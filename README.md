Jeeves
======


## Initial features (v0.0.0)
  - Only supports the promise chain version of wd
  - depends on:
    + `async`
    + `lodash`
    + `ensureDir`
    + `jQuery` (client-side)
    + `winston`

## Requirements/Setup
  - To use screenshots, a directory should be specified in the config otherwise by default they will save to `test-results/screenshots/`
